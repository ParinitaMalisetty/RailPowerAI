# RailPower AI  
### Machine Learning for Safety and Sustainability in Railways

---

## ğŸ“Œ Project Description

RailPower AI is a web-based machine learning system designed to enhance railway safety and energy sustainability.  
The project applies supervised machine learning techniques to analyze electrical and mechanical sensor data collected from railway systems.

The system focuses on:
- Fault classification in railway power systems  
- Load forecasting for efficient energy management  

By predicting faults and forecasting power demand, the system enables proactive maintenance, reduces downtime, and supports sustainable railway operations.
